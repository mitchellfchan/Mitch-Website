Rolling Out Public Artwork During Development and Construction Phase

It will be a long time before VMC is finished, but we want to get use art to draw people to the site now. We want to start branding the site as a cultural hub; we want to start making our connections with the art community. Typically, the public art is the last thing that you bring onto a construction site. But I expect that most of the units in this development are going to be sold pre-construction, and so if the art is going to increase the value of the development, you need to show art pre-construction. How do we start our art programming ASAP, taking on temporary projects at a lower cost?

Curatorial Vision

I want to be on some next-level shit right here. Let's buy young artists while we can still afford them, even if we don't install for another 5 years. Remember: when your taxi driver gives you a hot tip on a stock, it's time to sell that stock. Anything we're seeing in a shopping mall HAS TO GO. We need one and only one big flagship artist, like a Jeff Koons or a Richard Serra or maybe even an Anish Kapoor, in the $5M range, but for the most part, there's going to be no benefit to making a collection of the same old bullshit that you can see anywhere else.

A lot of this needs to be administration stuff for now

We need to really study the 1% for arts rule, and make some contacts about institutional collections to see if we can share resources. Going back to the timeline question, basically we need to know how much money we can spend and when we can spend it.

Fuck everything, let's make MARFA NORTH.
